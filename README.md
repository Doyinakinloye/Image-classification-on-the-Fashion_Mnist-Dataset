# Image-classification-on-the-Fashion_Mnist-Dataset

🧵 Fashion-MNIST Image Classification Project

This repository contains my work on image classification using the Fashion-MNIST dataset, a widely used benchmark dataset in machine learning and computer vision. The dataset consists of grayscale images of clothing items such as shirts, trousers, dresses, coats, sneakers, and more. The objective of this project was to train a machine learning model that can accurately predict the category of each clothing item based on its image.

🔑 Project Highlights

* Dataset Exploration & Preprocessing

* Loaded and examined the Fashion-MNIST dataset (70,000 images: 60,000 for training, 10,000 for testing).

* Normalized pixel values to improve training efficiency.

* Visualized sample images to understand the distribution of classes.

Model Development

* Implemented a classification model using standard machine learning techniques (without convolutional neural networks).

* Trained the model to distinguish between 10 clothing categories.

* Tuned hyperparameters to improve accuracy and reduce overfitting.

Evaluation & Results

* Measured performance using accuracy scores and confusion matrices.

* Visualized training progress with loss and accuracy curves.

* Achieved 89% accuracy on the classifcation, despite not implementing project with convolutional Neural network.

🚀 Tech Stack

* Python

* TensorFlow / Keras (for model building and training)

* NumPy, Pandas (for data handling)

* Matplotlib, Seaborn (for visualization)
